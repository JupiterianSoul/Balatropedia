# WASM Seed Finder Output

This directory contains the compiled WASM module from `wasm-seedfinder/`.

## Build

From the repo root:

```bash
bash wasm-seedfinder/build.sh
```

This requires Rust + wasm-pack. The build script auto-installs both if absent.

## Contents (after build)

- `seedfinder_bg.wasm` — compiled WASM binary (SIMD-enabled)
- `seedfinder.js` — ESM bindings generated by wasm-bindgen
- `seedfinder.d.ts` — TypeScript types

## Throughput targets

| Device | Target |
|--------|--------|
| Mid-2020 Android (Snapdragon 765G) | ≥ 50k seeds/sec |
| 2024 flagship desktop | ≥ 200k seeds/sec |

## Fallback

When `VITE_WASM_FINDER=0` or WASM SIMD is not supported by the browser,
the finder falls back to the existing JS+Immolate WASM worker.
